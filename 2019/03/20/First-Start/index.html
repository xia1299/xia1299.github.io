Hexo（node）+github创建自己的Bolg网站

发表于 2019-03-20 | 分类于 Hexo | 阅读次数：

前言

现在网上已经有很多关于如何使用Hexo+GitHub建立个人Blog网站的博客了，但是我还是决定自己写一篇，因为我想练练手！（哇哈哈哈）！由于MarkDown是第一次用，可能会很丑，不过也算是一个开端吧。

0.准备工作

如题所言，需要建立个人博客，需要做如下的准备：
软件
1. 安装Nodejs(因为Hexo是基于Nodejs的)
Windows安装 Nodejs 非常简单，我们只需要访问他的官网，然后下载安装就可以了。

2. 安装Git
Windows安装 Git,同样也是点击式安装。Git网站可能访问会比较慢，可以通过代理或者其他途径下载软件。
账户
确保自己有一个 github 的账户，没有的话就去申请一个！
